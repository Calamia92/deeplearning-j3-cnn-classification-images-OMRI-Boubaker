# Classification binaire d'images

Projet de l'apres-midi : comparer trois iterations sur un probleme de classification binaire d'images.

Dataset choisi : Cats vs Dogs Kaggle (`chetankv/dogs-cats-images`).

## Objectif

- TP1 : CNN from scratch
- TP2 : data augmentation + Dropout
- TP3 : transfer learning avec MobileNetV2

## Structure attendue des donnees

```text
data/
  train/
    classe_a/
    classe_b/
  val/
    classe_a/
    classe_b/
```

Les images ne sont pas versionnees dans Git. Seule la structure des dossiers est conservee.

## Installation

```bash
pip install -r requirements.txt
```
